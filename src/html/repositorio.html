Incidencias informáticas

En este proyecto, el objetivo es procesar y convertir datos en diferentes formatos. Primero, se debe obtener información de un formulario (como incidencias relacionadas con equipos informáticos), almacenándola en un archivo CSV. Luego, se realiza una conversión de esos datos a formato XML. Posteriormente, se crea un pequeño programa en Python para leer los datos en XML y procesarlos, mostrando la información relevante en pantalla. Finalmente, se almacena la misma información en formato JSON para demostrar la capacidad de manejar diferentes formatos de datos.

Validación de Edad

Este proyecto tiene como objetivo crear una herramienta en Python que permita validar si una persona es mayor de edad y puede trabajar. El usuario introduce su fecha de nacimiento, y el sistema calcula su edad en días. Luego, el sistema verifica si la persona tiene entre 16 y 65 años, ya que este es el rango de edad permitido para trabajar según el proyecto. Si la edad está dentro de este rango, el sistema muestra el mensaje "Pots treballar" (puedes trabajar), de lo contrario, muestra "No pots treballar" (no puedes trabajar). El sistema también maneja errores de entrada para garantizar que los datos introducidos sean válidos.

Proyecto3

Este programa tiene como objetivo calcular la cantidad de días que tiene un mes específico en un año determinado. Primero, solicita al usuario que ingrese un mes y un año en formato mes/año. Luego, valida que el mes sea un valor correcto (entre 1 y 12) y que el año no sea futuro. Una vez que los datos son válidos, el programa determina cuántos días tiene ese mes:
· Los meses con 31 días se calculan directamente.
· Los meses con 30 días se verifican.
· Para febrero, se comprueba si el año es bisiesto. Si lo es, febrero tiene 29 días; si no lo es, tiene 28 días.
De esta forma, el programa ayuda a determinar de manera precisa el número de días de cualquier mes de un año en particular, considerando también los años bisiestos.

Conversión de un Archivo CSV a XML con Python

Este programa convierte un archivo CSV a formato XML utilizando la librería pandas de Python. Primero, carga el archivo CSV especificado y limpia los nombres de las columnas, reemplazando espacios y eliminando caracteres especiales para asegurar que sean adecuados para el procesamiento. Luego, convierte el archivo CSV en un formato XML y guarda el resultado en un archivo con el nombre indicado. Finalmente, el programa imprime un mensaje confirmando que la conversión se ha realizado con éxito y el archivo XML se ha guardado correctamente. Esta solución es útil para transformar datos tabulares de un CSV a XML, un formato comúnmente usado para intercambio y almacenamiento de datos estructurados.
